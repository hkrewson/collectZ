# collectZ v3.4.78

## Event-local Social Editability Polish

This patch makes the Event drawer’s social sections easier to maintain in place. People, Groups, and Meetups can now be updated inline, so event-local coordination metadata stays close to the richer readback we added in the previous patch.

### What changed

- Added inline attendee editing for name, relationship, status, visibility, and notes.
- Added inline group editing for name, visibility, notes, and membership.
- Expanded meetup editing so group ownership and visibility can be updated alongside existing meetup details.
- Kept the existing Event-local backend model and patch endpoints intact.
- Kept friend-graph, cross-event identity, realtime presence, push/email/device delivery, and native companion UI work out of scope.
