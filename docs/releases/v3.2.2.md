## Summary
This patch release hardens release readiness evidence. It adds a repo-native local preflight helper that generates dependency audit artifacts and a go/no-go report from live stack evidence, and it remediates the `follow-redirects` dependency advisory in both backend and frontend package trees so the local dependency audit closes cleanly before release.

## What Changed
### Release evidence hardening
- Added a local release preflight helper that generates:
  - `artifacts/dependency-audit/backend-audit.json`
  - `artifacts/dependency-audit/frontend-audit.json`
  - `preflight-go-no-go.md`
- Made the local preflight explicitly distinguish:
  - passed local gates,
  - blocked local gates,
  - CI-only follow-through.
- Aligned the release checklist docs with the new preflight workflow.

### Dependency remediation
- Added package overrides so backend and frontend resolve `follow-redirects` to `1.16.0`.
- Refreshed both lockfiles so local dependency audits no longer report the previously tracked moderate advisory.
