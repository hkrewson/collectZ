## Summary

This patch continues the human-readable timeline work by making event-planning and Plex provider rows read like concrete activity instead of raw action names.

## What changed

- Added readable timeline copy for `events.*` rows, including attendees, groups, meetups, schedule plans, schedule imports, personal calendars, notifications, artifacts, and purchased-item links.
- Fixed the Events activity filter so event-planning rows logged as `events.*` appear under `Events`.
- Improved Plex timeline copy for rating readback/writeback, watched-state readback/writeback, refresh runs, reconciliation previews, sync jobs, and conflict resolution.
- Added browser coverage proving a workspace event attendee row appears as readable event activity.
- Hardened event/art/collectible Playwright cleanup helpers so authenticated deletes include CSRF tokens.
