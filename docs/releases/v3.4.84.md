# collectZ v3.4.84

## Event Catalog Mobile Time Window Filters

This patch continues the Event Schedule Catalog Now/Next follow-ups by making the mobile catalog snapshot easier to narrow during a con. The Now / Next catalog card now has compact mobile filters for all visible sessions, what is happening now, what is next, later today, and sessions already marked planned.

### What changed

- Added mobile-only time-window filters to the Event catalog Now / Next card.
- Supports `All`, `Now`, `Next`, `Later Today`, and `Planned` filters with visible counts.
- Keeps the default Now / Next readback unchanged until a mobile user chooses a filter.
- Preserves shared attendance, conflict readback, and quick plan-state actions in filtered rows.
- Keeps the full Schedule Catalog filter/edit surface intact for desktop and admin-heavy catalog management.
- Keeps native companion behavior, push/Discord/email delivery, cross-event identity, external contacts, realtime presence, and broader friend-graph work out of scope.
