## Summary

This patch makes Workspace Integrations explain whether each provider is using workspace-owned settings, collectZ defaults, or no usable configuration.

## What changed

- Added `integrationScope` readback to workspace integration responses.
- Workspace Integrations now shows a compact source badge for each provider section.
- Provider sections identify `Workspace`, `Default`, or `Not configured` beside the existing connection status.
- The source readback uses meaningful overrides such as saved keys, custom URLs, provider usernames, client IDs, Plex sections, CWA URL, and Kavita base URL.
