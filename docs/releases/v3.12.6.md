## Summary

This patch makes Dashboard Review drawers better at steering users toward title cleanup and assisted enrichment before asking for manual provider identifiers.

## What changed

- Added backend review clue fields for suggested lookup title, lookup context, and next action.
- Cleaned obvious file/import noise from assisted lookup titles before they reach Dashboard Review.
- Prefilled Dashboard Review lookup fields with backend review guidance when opening a row.
- Added a drawer action to copy the search text into the record title when the title itself is the likely problem.
- Kept manual identifier fields available as fallback controls.
- Updated OpenAPI and source-contract coverage for the new review guidance fields.

## Breaking changes

None.

## Environment/config changes

None.

## Migration and data impact

No schema migration or data backfill is required. Existing rows are unchanged until a user saves a title or metadata update from Dashboard Review.
