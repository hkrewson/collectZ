## Summary

This patch makes mobile library headers more compact by moving the visible page cue into the search row as a section icon while preserving the page title for accessibility.

## What changed

- Added icon-led mobile search rows to shared library headers.
- Added section icons for Audio, Art, Books, Comics, Collectibles, Events, Games, Loans, Movies, TV, and Capture Inbox.
- Kept page titles available as accessibility text.
- Changed the mobile app shell title to screen-reader-only text so it no longer duplicates the page cue visually.
- Preserved desktop title/header behavior.
