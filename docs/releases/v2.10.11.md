## Summary
This patch release tones down the app’s remaining heavy overlay shells so the camera modal and the main slide-in drawers feel calmer and more functional. The overlays still focus attention and behave the same way, but they no longer rely as heavily on blur, oversized rounding, or dramatic shell/poster shadowing to create that sense of separation.

## What changed
### Modal shell flattening
- Replaced blur-heavy dark backdrops in the shared camera modal and the main slide-in overlays with simpler scrim-style backdrops.
- Reduced shell weight by replacing the deeper shadow treatment on the main overlay shells with calmer card-level shadowing.
- Reduced the camera modal shell corner radius from `rounded-2xl` to `rounded-xl`.

### Drawer shell alignment
- Softened the floating poster treatment inside the library and collectibles detail drawers by replacing the deeper shadow treatment with the calmer shared card shadow.
- Flattened the add/edit and detail overlay shells in the library, events, and collectibles surfaces without changing dismissal, layout, or editing flows.

## Breaking changes
- No backend or API-breaking changes are introduced in `2.10.11`.
- This release intentionally changes shared overlay presentation to be calmer and less staged.
