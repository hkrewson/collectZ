## Summary

This patch stabilizes the admin shell browser regression in CI by making the test setup enable the gated library surfaces it asserts against and by avoiding brittle browser response-body reads for dashboard summary data.

## What changed

- Browser setup now enables both `events_enabled` and `collectibles_enabled`.
- The dashboard command center regression reads `/api/dashboard/summary` through the authenticated Playwright request context instead of parsing a page network response body.
- The mobile compact toolbar and desktop section-heading regressions now run with Art, Collectibles, and Events available in the CI fixture state.
