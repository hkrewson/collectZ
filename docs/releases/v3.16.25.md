## Summary

This patch fixes a homelab edition shell leak where admin users could still see Platform navigation artifacts after the workspace/platform navigation split.

## What changed

- Gated Platform sidebar navigation behind the platform product edition instead of admin role alone.
- Kept homelab admin settings and integrations available as local `Settings` and `Integrations` links.
- Updated homelab settings copy so direct settings routes no longer render `Platform Settings`.
- Made the mobile shell title edition-aware for homelab settings and integrations.
- Added browser regression coverage proving homelab admins do not see Platform, Platform Settings, Runtime, or the account-menu Platform switch.
