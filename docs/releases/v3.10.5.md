# collectZ v3.10.5

## Version and date

- Version: `3.10.5`

- Milestone: `3.10.5 — Compact Mobile App Header`

## Milestone target and status

- Status: `completed`

## Summary

This patch replaces the mobile brand-style app header with a compact utility header that keeps navigation accessible while giving the space back to the active page.

## What changed

- Replaced the mobile hamburger button with the collectZ mark as the navigation trigger.
- Kept the navigation trigger accessible with the same open-navigation behavior and expanded-state readback.
- Removed the mobile `COLLECTZ` wordmark and role/workspace readback from the app header.
- Added active page title readback in the mobile app header, including Dashboard, Loans, Wishlist, Import, and Integrations.
- Updated the sidebar wordmark casing from `COLLECTZ` to `collectZ`.
- Extended mobile browser regression coverage for the compact app header and nav trigger.

## Breaking changes

None.

## Environment/config changes

None.

## Migration and data impact

No schema migration or data backfill is required. This patch changes frontend shell layout, browser regression coverage, version metadata, and release notes only.
