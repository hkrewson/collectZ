## Summary

This patch keeps the Workspace and Platform split moving in the right direction by removing the old collapsible Workspace navigation group and making Help versus Help Admin clearer.

## What changed

- Removed the collapsible `Workspace` navigation item.
- Promoted workspace `Settings` and `Merge Review` to direct Workspace-mode navigation links.
- Kept normal users in `Help` from the Workspace menu.
- Moved `Help Admin` into the platform/support navigation context instead of showing it as a Workspace item.
- Updated browser and unit coverage so the sidebar contract expects the flattened Workspace links and platform-scoped Help Admin behavior.
