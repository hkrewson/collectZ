# collectZ v3.4.130

## Summary

This patch adds an opt-in Plex watched-state refresh scheduler for existing Plex-linked collectZ rows.

## Changes

- Added admin status and manual-run endpoints for Plex watched-state refresh automation.
- Added environment controls for the scheduler timer and batch size.
- Reused the existing watched-state apply path so scheduled refreshes update only matched movie metadata and TV season state.
- Added fake-PMS smoke proof that the refresh sweep does not create new media rows.
- Kept Plex scrobble/unscrobble writeback, rating writeback apply behavior, full-library reconciliation, and broad import rewrites out of scope.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- Smoke evidence redacts Plex tokens, token query strings, private IP addresses, and raw media file paths.
