# v3.4.33

- Date: 2026-04-29
- Version: `3.4.33`
- Target milestone: `3.4.33 — Event Schedule Agenda Drawer Polish`

## Summary

This patch makes Event schedule plans easier to read in the detail drawer. The schedule now appears before feed and social management, and planned sessions are shown as a compact day-grouped agenda instead of a stack of large cards.

## User-Facing Changes

- Event detail drawers now prioritize the schedule section when schedule plans exist.
- Schedule plans are grouped by day with compact time, title, location, category, and source metadata.
- Session descriptions, source links, and remove actions are available by expanding a row.
- Sched feed controls are tucked into a separate `Manage Sched feed` section.
- People, Groups, and Meetups remain available as secondary event-planning sections.

## Validation Notes

- This is a UI readability patch only.
- No continued Sched syncing, full schedule catalog import, or Now/Next discovery behavior was added.
- Docker-first frontend rebuild should report `3.4.33` from `/api/health`.
