## Artwork Numbered Print Metadata and Badge

## Summary

This patch adds item-local numbered print metadata to Art so limited prints can show their specific edition signal while browsing and in the detail drawer.

## What Changed

### Shipped

- Added positive-integer Art metadata fields for print number and print run.
- Added editor inputs for partial or complete numbered print values.
- Added clean card subtitle readback plus list, mobile detail, and desktop detail readback for numbered prints.
- Shows concise copy such as `#12/100 Signed Print`, `Print #12`, or `Print Run 100` when print metadata exists.
- Preserved non-numbered artwork entry with no new required fields.
- Reduced the local Graylog evidence stack's default journal size so release observability checks can run in constrained Docker Desktop storage.
