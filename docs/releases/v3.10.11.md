## Summary

This patch tightens the Capture Inbox header so the page keeps more usable space on mobile and uses the same compact action language as the rest of the current interface pass.

## What changed

- Removed the library subtitle from the Capture Inbox page header.
- Kept `New capture` as an always-icon camera button.
- Kept `Batch scan` as an always-icon barcode button.
- Replaced separate capture/source/review controls with one compact filter button.
- Added a shared funnel-style icon to the app icon set.
- Updated browser regression coverage for the compact Capture Inbox header and filter menu.

None.
