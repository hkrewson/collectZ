# collectZ v3.4.131

## Summary

This patch adds a safe Plex rating readback apply path for existing collectZ rows.

## Changes

- Added an admin-only Plex rating apply endpoint.
- Reads Plex `userRating` from metadata readback and applies it to matching collectZ `user_rating`.
- Stores lightweight Plex rating metadata for readback/provenance.
- Added fake-PMS smoke proof that rating apply does not create new media rows.
- Kept collectZ-to-Plex `/:/rate` writeback, watched-state writeback, scheduled rating refresh, full-library reconciliation, and broad import rewrites out of scope.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- Smoke evidence redacts Plex tokens, token query strings, private IP addresses, and raw media file paths.
