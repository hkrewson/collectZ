## Kavita Embedded Reader Controls Polish

## Summary

This patch polishes the Kavita chapter reader controls so page preview, progress readback, save progress, mark read, and reset progress feel like one coherent drawer workflow.

## What Changed

### Shipped

- Renamed the drawer section to `Kavita Reader` for chapter-backed Kavita rows.
- Added reader status badges for loaded state, current page, total pages, and saved progress.
- Switched drawer page entry to user-facing one-based page numbers while preserving zero-based Kavita API calls behind the scenes.
- Added page-image loading and error states for the proxied reader image.
- Grouped `Load Reader`, `Save Progress`, `Reset Progress`, and `Mark Read in Kavita` beside the page preview.
- Kept reset-progress copy separate from true unread behavior.
