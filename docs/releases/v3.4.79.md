# collectZ v3.4.79

## Event Self-Attendee Auto-Link and Add Me Flow

This patch smooths out one of the rougher spots in Event social planning. Instead of asking the signed-in user to manually mark an attendee row as their app identity, the People panel now offers a direct `Add me` path and leaves the generic attendee form focused on adding other people.

### What changed

- Replaced the old self-link checkbox in the Event drawer People form with an explicit `Add me` action for the signed-in user.
- Auto-fills the self-attendee name from the current app user when available.
- Keeps the regular People form available for adding other attendees.
- Preserves the existing Event-local attendee ownership model and linked-user readback.
- Keeps external contact identities, cross-event identity, native companion mutation UX, and broader friend-graph work out of scope.
