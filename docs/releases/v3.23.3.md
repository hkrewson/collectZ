## Summary

This release continues the maintainability guardrails work by cleaning shared app primitives before the next large-view extraction pass. It removes shared accessibility lint failures around overlays, pagination labels, and clickable poster cards, and trims React hook reset-effect noise in shared drawer/media primitives.

## What changed

- Replaced clickable overlay `div` elements in shared drawer and camera primitives with native buttons.
- Associated the shared pagination page-size label with its select control.
- Made interactive poster cards render with native button semantics when they are clickable or selection-aware.
- Derived cover-image preview URLs and drawer overview expanded state without reset effects.
- Added a small keyed-draft helper so collectible metadata editors can reset draft state when their owner/trait key changes without synchronous reset effects.
- Documented narrow external-system effect exceptions for mobile shell DOM/media-query synchronization and relationship search/reset state.
- Reduced the full frontend quality baseline from `170` errors / `65` warnings to `153` errors / `61` warnings.
