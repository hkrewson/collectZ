# collectZ v3.20.7

Plex readback refresh is now visible from the Plex Sync workflow.

## Highlights

- Expanded the Plex watched-state refresh pass to also apply Plex user-rating readback.
- Added readback refresh status to Settings > Integrations > Plex > Sync.
- Added an admin manual Refresh readback action.
- Kept Plex writeback separate and gated behind explicit opt-in controls.

## Notes

This is still read-only toward Plex. It pulls watched state and ratings from Plex into collectZ; it does not push local changes back to Plex.
