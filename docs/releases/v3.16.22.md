- Added unit coverage that keeps media identity alias SQL on query parameters.
- Replaced legacy AudioDB URL substring detection with parsed-hostname matching.
- Replaced broad source-link hostname substring checks in the Library detail drawer with parsed hostname label matching.
- Updated the PriceCharting unit-test mock to compare parsed hostnames instead of broad URL substrings.
