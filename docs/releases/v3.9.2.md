## Version and date

- Version: `3.9.2`

- Milestone: `3.9.2 — Apple/iTunes Wishlist Price History Foundation`

## Milestone target and status

Status: Complete.

This patch stores successful Apple/iTunes wishlist price refreshes as scoped history snapshots and adds a compact readback in Library > Wishlist.

## Changes

- Added `wanted_item_price_history` for scoped wishlist price snapshots.
- Writes a price-history row when manual Apple/iTunes price refresh successfully reads a saved item.
- Added authenticated `GET /api/wishlist/{id}/price-history`.
- Added a compact `Price history` row action for Apple-linked wishlist rows.
- Documented the price-history route and snapshot response shape in OpenAPI.

## Security and privacy

- Price history remains authenticated and scoped through the existing Wishlist access controls.
- Raw Apple lookup context is stored only in scoped snapshot metadata.
- No scheduled polling or notification delivery is introduced in this slice.
