## Summary

This release continues the maintainability guardrails path by reducing the highest-signal frontend ESLint baseline in a targeted way. It cleans up Sidebar navigation helper components and adds React JSX usage lint support so JSX-only component references are not falsely reported as unused.

## What changed

- Hoisted Sidebar helper components out of render to satisfy React hooks static-component checks.
- Made Sidebar backdrop and collapsed-rail interactions satisfy keyboard/pointer accessibility lint expectations.
- Removed unused Sidebar destructuring for no-longer-used space props and derived platform-group state.
- Added `eslint-plugin-react` with `react/jsx-uses-vars` so JSX component usage is recognized by ESLint.
- Reduced the full frontend quality baseline from `200` errors / `356` warnings to `170` errors / `65` warnings.
