# collectZ v3.4.120

Release date: 2026-05-07

## Summary

This release adds saved Plex Now Playing display preferences. Admins can tune the passive display surface for `/now-playing` and display-token links without changing Plex imports, webhooks, scheduled sync, or watch-state behavior.

## Changes

- Added stored Plex Now Playing display preferences to integration settings.
- Added admin controls for poster, backdrop, context, player, progress, refresh time, paused sessions, and compact/standard/large text scale.
- Returned normalized display preferences from both the admin-session viewer and display-token viewer endpoints.
- Updated `/now-playing` to respect those preferences in both authenticated and display-token modes.
- Kept Plex imports, token generation/revoke behavior, image proxying, webhooks, scheduled sync cadence, and watch-state work unchanged.
