## Summary

This patch corrects the mobile header icon placement so the current section icon lives in the top app header navigation button instead of taking space in the search toolbar row.

## What changed

- Moved mobile section icons into the shell header navigation button.
- Removed the section icon slot from the shared mobile search toolbar.
- Kept mobile page titles available as screen-reader text and in the navigation button label.
- Updated Capture Inbox mobile readback to use the full page name.
- Preserved desktop header behavior and existing search/filter/add toolbar behavior.

## Breaking changes

None.

None.
