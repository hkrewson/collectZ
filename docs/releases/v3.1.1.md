## Summary
This patch release closes the confidence gap between the shared-core lifecycle work from `3.1.0` and what users actually see in the browser. The Playwright regression layer now proves browser-visible fallback and boundary behavior for support sessions, library lifecycle fallout, and space lifecycle fallout, so the extracted shared-core repair paths are no longer validated only by backend/runtime smokes.

## What changed
### Browser-visible support-session lifecycle proof
- Added Playwright coverage for approved support-session flows where:
  - support-session state becomes browser-visible in the shell
  - stale support-library session drift is normalized after reload
  - teardown removes temporary support-session controls and restores the expected admin/help boundary

### Browser-visible library lifecycle fallback proof
- Added Playwright coverage for browser-visible fallback when:
  - the active library is archived
  - the active library is transferred away from the previous owner
- The browser regression now proves that surviving accessible-library context is restored cleanly in shell-visible library/import flows.

### Browser-visible space lifecycle boundary proof
- Expanded Playwright coverage for affected members so the browser now proves:
  - suspended members lose workspace affordances and hit the access-restricted boundary
  - restored members regain workspace access
  - removed members remain blocked from workspace management surfaces

### Browser-suite stabilization aligned to the current platform contract
- Tightened older browser assertions that had drifted from the live platform behavior:
  - integrations specs now sign in explicitly instead of relying on saved auth bootstrap state
  - approved support-session teardown assertions now focus on the real post-session contract rather than a brittle nav assumption
  - one space lifecycle assertion now checks the access-restricted boundary contract instead of exact explanatory copy

## Breaking changes
- None intended for operators or end users.
- `3.1.1` is a regression-expansion and verification release only.
