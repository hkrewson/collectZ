- Roadmap slice: `3.19.5 — Saved Views Navigation`

## Summary

This patch adds a Library navigation entry point for saved views. Users can open Saved Views from the Library navigation, review their durable saved Library views grouped by media type, and jump into the matching Library with the saved view applied.

## What changed

- Added `Saved Views` to the Library navigation group.
- Added a saved Library views index surface.
- Wired saved-view selections into the matching media Library tab.
- Added a `LibraryView` handoff prop so navigation-selected saved views apply after saved views load.
- Added unit/source coverage for the saved-view navigation contract.

## Breaking changes

- None.

## Environment/config changes
