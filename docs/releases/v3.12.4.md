## Summary

This patch makes Dashboard Review Missing IDs drawers more specific to the item being reviewed instead of showing the same generic search controls for every media type.

## What changed

- Added type-specific lookup guidance in Dashboard Review drawers.
- Movie and TV review rows now keep TMDB search with year context.
- Book review rows now show Google Books search with author context.
- Comic review rows now show comic issue search without a misleading year field.
- Audio review rows now show Discogs search with artist context.
- Game review rows now show game search with platform context when available.
- Kept manual identifier fields available as fallback controls.
- Updated browser and unit source-contract coverage for the drawer search surface.

## Breaking changes

None.

## Environment/config changes

None.

## Migration and data impact

No schema migration or data backfill is required. Existing media rows are unchanged until a user applies a match or saves edits from the review drawer.
