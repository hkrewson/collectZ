## Milestone target and status

Status: Complete.

This patch adds a conservative, opt-in scheduler path for Apple/iTunes wishlist price refreshes.

## Changes

- Added backend runtime config for Apple/iTunes wishlist price refresh automation.
- Added authenticated admin scheduler status and manual-run endpoints.
- Scheduler defaults off and can be enabled by environment configuration.
- Manual scheduler runs reuse the same price refresh and history snapshot path as the existing Wishlist refresh.
- Added compact Wishlist readback for auto-refresh status and a `Run auto refresh now` action.
- Documented the scheduler status/run routes in OpenAPI.

## Security and privacy

- Scheduler inspection and manual runs are admin-only.
- Recurring polling defaults off.
- No price-drop alerts, notifications, charts, or owned-library valuation behavior are introduced.
- Apple raw lookup context remains scoped to wishlist item metadata and price history snapshots.
