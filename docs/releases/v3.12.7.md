## Summary

This patch makes Dashboard Review defer/dismiss decisions visible and reversible from the same Review panel, without adding a standalone Review page.

## What changed

- Added active hidden review decisions to the Dashboard summary payload.
- Added a scoped restore endpoint for Dashboard Review defer/dismiss decisions.
- Added a compact collapsed `Hidden review items` section inside Dashboard Review.
- Restoring a hidden item removes the decision and refreshes the Dashboard summary.
- Added OpenAPI and source-contract coverage for hidden decision readback and restore behavior.

## Breaking changes

None.

## Environment/config changes

None.

## Migration and data impact

No schema migration or data backfill is required. Existing `media_review_decisions` rows are read by the Dashboard summary. Restoring a hidden item deletes the selected decision row only.

Rollback is safe. If hidden decision readback behaves unexpectedly, roll back to `3.12.6`; existing decision rows remain compatible with `3.12.5+` defer/dismiss behavior.

## Known issues and follow-up

- This patch restores decisions one at a time.
- It does not add bulk restore or a full maintenance history view.
- Standalone Review pages/nav items remain out of scope unless a future workflow truly cannot live in Dashboard Review.
