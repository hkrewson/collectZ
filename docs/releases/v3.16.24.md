## Summary

This patch continues CodeQL baseline remediation by replacing risky sanitizer and regex helper patterns with deterministic normalization, encoded image URLs, and runtime-generated smoke-test credentials.

## What changed

- Hardened shared frontend image URL handling so art and collectible image previews only render explicit safe image URL forms and encode DOM-bound image sources.
- Replaced repeated-period review clue cleanup with a complete repeated-period collapse.
- Replaced Kavita and request-origin slash trimming regexes with deterministic slash trimming helpers.
- Replaced support issue-link and collectible trait-key regex normalization with bounded scanners.
- Replaced automatic loan reminder smoke-test randomness and hardcoded transient password material with crypto-generated runtime values.
- Moved personal Sched ICS `&amp;` decoding after other entity replacements to prevent double-unescape behavior.
- Extracted support summary state/loading from the app shell so `frontend/src/App.jsx` stays under the CI hard line budget.
- Updated browser regression coverage for the current Platform Runtime and Workspace Integrations split so CI no longer expects stale provider tabs on the platform runtime surface.
