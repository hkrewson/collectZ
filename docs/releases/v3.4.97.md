## Kavita Metadata Writeback Preview and Diff

## Summary

This patch adds the first read-only Kavita metadata writeback preview path. Workspace admins can compare current Kavita metadata with collectZ metadata for a Kavita-linked row before any later apply/writeback milestone exists.

## What Changed

### Shipped

- Added a workspace-admin-only Kavita metadata preview endpoint for Kavita-linked media rows.
- Added current Kavita series metadata readback through `GET /api/Series/metadata` and chapter preview readback through stored Kavita volume/chapter details.
- Added field-level diff output with changed, unchanged, skipped, locked-field, and preview-only mutation-disabled readback.
- Added a read-only Kavita Metadata panel in the media detail drawer so operators can inspect preview diffs without writing to Kavita.
- Extended the Kavita import/sync smoke to prove preview-only behavior, locked-field skipping, workspace-scoped credentials, and secret-free preview responses.
