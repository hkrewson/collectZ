## Summary

This patch improves the Event drawer's personal Sched feed status so sync failures, stale feeds, and never-synced feeds are easier to understand without making the selected schedule feel broken.

## User-Facing Changes

- `Manage Sched feed` now shows clearer collapsed status such as synced, stale, not synced, syncing, or needs attention.
- Expanded feed details distinguish the last successful sync from the last refresh attempt.
- Failed refreshes explain that previously synced schedule items remain usable when available.
- The personal Sched iCal URL remains hidden and is never shown back in the drawer.
