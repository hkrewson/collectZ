# v3.4.68

## Event schedule notification draft management UI

- Version: `3.4.68`
- Date: `2026-05-02`
- Target milestone: `3.4.68 — Event Schedule Notification Draft Management UI`

### Summary

This release makes Event-local schedule notification drafts manageable after they are saved.

### Changes

- Added draft-only update and discard endpoints for Event-local schedule notifications.
- Added `Edit draft`, `Send draft`, and `Discard draft` actions to schedule notification history.
- Let saved drafts rehydrate the composer with message text and selected recipients.
- Let users send the same saved draft record instead of creating a separate replacement notice.
- Kept sent records immutable from the drawer draft-management UI.
- Kept push, email, native device delivery, global inboxes, friend graphs, and realtime social behavior out of scope.
