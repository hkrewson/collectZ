- Roadmap slice: `3.12.14 — Dashboard Standard Tabbed Layout`
- Status: completed

## Summary

This patch makes the Dashboard use its compact tabbed layout at every width, so wide screens no longer spread Review, Syncs, Activity, Health, and Events into separate panels.

## What changed

- Dashboard section tabs are now the standard layout on desktop and mobile.
- The wide-screen multi-panel Dashboard layout was removed.
- Dashboard Review lists can now show more sample rows before users need to drill into another surface.
- Existing Dashboard Review drawer behavior, provider health, recent syncs, recent activity, and upcoming events content are unchanged.

## Breaking changes

- None.

## Environment/config changes

## Migration and data impact

- Existing Dashboard summary data and review decisions are unchanged.
