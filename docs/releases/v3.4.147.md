# collectZ v3.4.147

- Version: `3.4.147`
- Status: `released`

## Summary

This patch adds a read-only real Plex Media Server provider item-row parity proof. It uses saved Plex settings, probes provider-advertised item listing candidates with small page limits, and records whether those rows carry enough metadata to consider a future import migration.

## Changes

- Adds a real-PMS proof script for provider-advertised item row parity.
- Adds a service helper that reads small pages from sanitized provider item-listing candidates.
- Records field-coverage counts for rating keys, provider GUIDs, TMDB IDs, artwork keys, variant resolution, season counts, watched state, and user ratings.
- Keeps Plex import, reconciliation, watched-state writeback, rating writeback, and automatic sync behavior unchanged.
- Keeps legacy Plex import current until full real-server parity and repeat-sync safety are proven.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- This proof writes only sanitized counts and booleans; it does not persist Plex tokens, provider URLs, private IPs, machine identifiers, media paths, or raw PMS payloads.
