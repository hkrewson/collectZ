- Version: `3.9.11`

## Milestone target and status

Status: complete.

This patch improves the Apple/iTunes Wishlist search loop. Search results that are already saved now read back the saved Wishlist status, and saved candidates expose a `View saved item` action so the user can jump the Wishlist list to the matching saved row instead of staring at a disabled button.

## Changes

- Apple/iTunes search candidates now include saved Wishlist status when a provider match already exists.
- Saved Apple/iTunes search results show readable copy such as `Saved as Watching`.
- Saved search results now offer `View saved item`, which filters the Wishlist list to the saved provider key.
- Newly saved search results update in place with `Saved as Wanted`.
