# collectZ v3.4.148

- Version: `3.4.148`
- Status: `released`

## Summary

This patch polishes the Plex Now Playing display for homes with more than one active Plex session. The main display still focuses on the first active session, and can now show a compact readback of other active sessions when that display preference is enabled.

## Changes

- Adds an `Other sessions` display preference to the Plex integration settings.
- Shows up to four additional active sessions on the standard Now Playing display.
- Keeps the vertical poster-only layout poster-only.
- Keeps Plex import, reconciliation, webhooks, watched-state writeback, rating writeback, and automatic sync behavior unchanged.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.

- This UI polish does not expose Plex tokens, provider URLs, private IPs, machine identifiers, media paths, or raw PMS payloads.
