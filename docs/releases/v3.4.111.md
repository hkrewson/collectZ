## Summary

This release starts the Plex PMS modernization path without changing existing Plex import behavior. The current Plex sync remains on the proven legacy library paths, while new Plex-facing work now has a documented provider-oriented contract and a small `/media/providers` discovery parser to build on.

## What changed

### Backend

- Added a Plex PMS modernization contract that records current legacy import paths and the intended provider-oriented discovery seam.
- Added lightweight parser/normalizer support for PMS media-provider discovery payloads.
- Added source assertions for the contract, parser behavior, docs, and backlog promotion.
- Preserved existing Plex import, duplicate-avoidance, TV season, and provider identity behavior.

### Frontend

- No frontend runtime UI behavior changed.
- Version metadata now points Help and health surfaces at `v3.4.111`.

### Database
