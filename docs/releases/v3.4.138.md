# collectZ v3.4.138

## Summary

This patch normalizes collectZ user ratings to a 0-10 internal scale while preserving the existing 5-star UI.

## Changes

- Stores `user_rating` on a provider-compatible 0-10 scale.
- Keeps the visible star controls as 0-5 stars with half-star selection.
- Converts star clicks at the UI boundary before saving.
- Allows Plex rating writeback to send the stored 0-10 value directly to Plex.
- Keeps Plex rating readback on the same 0-10 internal scale.
- Adds a one-time migration for existing 0-5 star ratings.
- Extends Plex writeback and browser assertions so 3.5 stars maps to Plex 7/10 instead of 3.5/10.
