## Summary

This patch stabilizes the mobile app header so the CollectZ/menu bar stays in place while library content scrolls underneath it.

## What changed

- Changed the dashboard shell to use dynamic viewport height on mobile-oriented browsers.
- Made the mobile app header sticky with a stable z-index.
- Preserved the existing mobile header visuals, navigation behavior, and library page toolbars.
- Added browser regression coverage that scrolls mobile library pages and verifies the app header does not move or create horizontal overflow.

## Breaking changes

None.

## Environment/config changes

None.

No schema migration or data backfill is required. This patch changes frontend shell layout, browser regression coverage, version metadata, and release notes only.
