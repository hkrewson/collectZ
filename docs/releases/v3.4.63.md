## Event schedule shared attendance on session cards

- Version: `3.4.63`
- Date: `2026-05-02`
- Target milestone: `3.4.63 — Event Schedule Shared Attendance on Session Cards`

### Summary

This release makes shared Event schedule context easier to see by showing visibility-safe people/group readback directly on session cards.

### Changes

- Added compact shared-attendance text to Now/Next, catalog session, and personal schedule rows.
- Prefer already-visible Event attendee and group names, such as `Shared with Reid, Panel crew`, when that context exists.
- Kept the existing count-based fallback, such as `Shared: 1 backup`, for shared sessions without attendee/group names.
- Updated personal schedule rows so changing draft visibility immediately updates the shared-attendance readback.
- Added browser/unit coverage for the shared-attendance card behavior.
