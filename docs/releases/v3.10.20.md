## Summary

This patch quiets the right side of desktop library headers by making secondary controls more compact while keeping search, filter, view, sort, and add actions available.

## What changed

- Tightened the shared library header view toggle spacing.
- Made the desktop add action icon-only while preserving accessible labels and titles.
- Grouped view/sort controls more tightly so search and filter remain the primary desktop header controls.
- Kept the existing mobile compact header behavior intact.
