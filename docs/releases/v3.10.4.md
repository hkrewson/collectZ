## Summary

This patch brings Loans onto the fixed-header page pattern added for the other utility pages in `3.10.3`.

## What changed

- Converted Loans to use the shared `FixedPageShell` primitive.
- Kept the Loans title, search, status filters, and summary counters stationary while loan rows scroll underneath.
- Extended the mobile utility-page browser regression so Loans is covered with Wishlist, Import, and Admin > Integrations.

## Breaking changes

None.

## Environment/config changes

None.

## Migration and data impact

No schema migration or data backfill is required. This patch changes frontend layout ownership, browser regression coverage, version metadata, and release notes only.
