# collectZ v3.4.134

## Summary

This patch adds explicit Plex rating writeback for one existing Plex-linked collectZ row.

## Changes

- Added an admin-only rating writeback endpoint for `PUT /:/rate`.
- Resolves the target from an existing Plex-linked collectZ row by media id or Plex rating key.
- Writes the collectZ rating to Plex using `identifier=com.plexapp.plugins.library`, the Plex rating key, and a 0-10 rating.
- Stores lightweight writeback provenance metadata on the collectZ row.
- Keeps UI auto-sync, scheduled rating writeback, full-library reconciliation, and broad import rewrites out of scope.
