# collectZ v3.8.14

## Version and date

- Version: `3.8.14`
- Date: `2026-05-19`

## Milestone target and status

- Milestone: `3.8.14 — Capture Safe Exact ISBN Auto-Add`
- Status: `implemented`

## Summary

This release makes batch scanning faster when Capture Inbox finds one safe exact ISBN book match. Multiple lookup candidates still stay in the editor for user choice.

## What changed

- Added safe exact ISBN match detection to Capture Inbox.
- Batch scan now automatically saves and imports a single exact ISBN book match.
- Normal one-off captures still show the candidate list in place.
- Single exact ISBN rows now use clearer `Add exact ISBN` / `Link exact ISBN` copy.
- Kept multi-candidate ISBN/UPC lookups as explicit user choice.

## Breaking changes

- None.

## Environment/config changes

- No new required environment variables.

## Migration and data impact

- No database migration.
- Auto-add uses the existing capture save and selected-match import path.

## Security triage

- Capture lookup and import remain authenticated and scoped.
- No provider credentials or scanner secrets are exposed.

## Security vulnerability triage

- High findings: none identified in local checks for this slice.
