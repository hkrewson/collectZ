## Summary
This patch release takes the last clearly isolated low-risk Dependabot PR from the open `2.7.x` queue: the direct frontend `postcss` bump. It refreshes the direct frontend CSS tooling dependency without attempting to force the remaining CRA-owned `resolve-url-loader -> postcss` advisory path through another patch-level workaround.

- Updated the direct frontend `postcss` dependency to `8.5.8`.
- Regenerated the frontend lockfile so clean installs resolve the direct frontend `postcss` tree to `8.5.8`.
