# collectZ v3.20.1

## Version and date

- Version: `3.20.1`
- Date: 2026-06-20

## Milestone target and status

- Roadmap slice: `3.20.1 - Plex Sync Cadence Settings`
- Status: completed

## Summary

This patch adds persisted Plex reconciliation sync settings. The Plex Sync tab can save automatic sync enablement, interval, and default scan limit, and scheduler readback uses stored settings when runtime env does not override them.

## What changed

- Added migration `111` for Plex reconciliation sync settings.
- Added `plexReconciliationSyncSettings` to integration settings payloads.
- Saved Plex URL, selected library sections, key updates, and sync cadence through the same Plex settings save path.
- Added compact automatic sync controls to Settings > Integrations > Plex > Sync.
- Updated scheduler readback and manual scheduler runs to prefer stored sync settings unless env overrides are configured.
- Preserved existing manual sync, queued preview, conflict review, and writeback behavior.

## Breaking changes

- None.

## Environment/config changes

- Existing `PLEX_RECONCILIATION_SYNC_*` env vars still override stored settings.
- Without env overrides, stored Plex sync settings are used.

## Migration and data impact

- Adds nullable/defaulted columns to `app_integrations`.
- Existing installs default automatic Plex reconciliation sync to off with a 360-minute interval.

## Deployment and verification

- Rebuild backend and frontend containers with `APP_VERSION=3.20.1`.
- Verified locally:
  - `node --check backend/routes/integrations.js`
  - `node --check backend/routes/media.js`
  - `node --check backend/services/integrations.js`
  - `node --check backend/db/migrations.js`

  - `node backend/scripts/unit-tests.js`

## Rollback guidance

- Roll back to the previous backend and frontend images if Plex integration settings or scheduler readback regress.
- No destructive migration rollback is required; the added columns can remain unused.

## Known issues and follow-up

- This patch does not add webhook reachability validation.
- Activity entries for automatic scheduled sync outcomes remain a follow-up in the Plex true-sync plan.
