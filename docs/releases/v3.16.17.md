## Summary

This patch refines sidebar navigation feedback so hover and active states use compact blue underlines instead of gray row highlighting.

## What changed

- Removed muted gray hover backgrounds from sidebar navigation items.
- Added a muted blue hover/focus underline that appears only under the hovered or focused item.
- Shortened sidebar underlines so they read closer to the label width instead of spanning the full nav row.
- Kept active selected items on a solid blue underline.
- Preserved the collapsed-rail active indicator for grouped Library and Platform navigation.
- Updated unit coverage for the hover underline contract.
