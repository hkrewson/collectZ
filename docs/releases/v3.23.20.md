## Summary

This patch removes signed, numbered-print, and event-acquisition pills from Art poster cards. Cards now keep the artwork and title dominant with at most two quiet plain-text metadata lines.

## What changed

- Replaced the combined numbered/signed/medium subtitle with a plain `medium · edition` line.
- Added one optional plain signer line using `Signed by artist`.
- Kept `Signed` on the first line only when an item is signed but no signer name is known.
- Removed collectible trait pills, including event/vendor/booth context, from Art poster cards.
- Kept complete collectible and acquisition metadata in existing list and detail surfaces.
- Extended mobile browser coverage to verify the two-line contract, absence of trait/event pills, and viewport containment.
