# collectZ v3.4.95

## Kavita Workspace-Owned Integration Administration Implementation

This release implements the workspace-owned Kavita administration contract from `3.4.94`.

### Shipped

- Added workspace-owned Kavita settings to Space Manager integrations.
- Added workspace-scoped Kavita save, test, clear, import, and chapter fan-out behavior.
- Required workspace-admin access before Kavita imports can run.
- Stopped Kavita import and cover proxy execution from falling back to legacy platform-level Kavita credentials.
- Kept Kavita settings readback redacted.
- Extended Kavita import smoke coverage for overlapping Kavita ids across workspaces and scoped cover readback.

### Boundaries

- Kavita remains read-only.
- Embedded reading, reader page proxying, progress sync, metadata writeback, special-chapter import, and shared provider abstractions remain future work.
- Legacy platform-level Kavita settings remain compatibility data, not the primary workspace administration path.
