# collectZ v3.4.135

## Summary

This patch adds explicit Plex writeback controls to Plex-linked media detail drawers.

## Changes

- Exposes Plex-linked media state in list and detail readbacks.
- Shows admin-only Plex controls for linked rows in the media detail drawer.
- Adds a manual `Push rating to Plex` action that uses the existing rating writeback endpoint.
- Adds manual watched/unwatched Plex actions for movie and non-TV rows.
- Keeps TV-series watched-state writeback blocked until a later episode-aware slice.
- Keeps automatic writeback, scheduled writeback, full-library reconciliation, and broad Plex import rewrites out of scope.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- Plex writeback controls remain admin-only and rely on existing backend authorization for mutation.
