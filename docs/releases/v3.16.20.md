## Summary

This patch keeps the collapsed sidebar brand mark aligned with the navigation icon column. It continues the 3.16 sidebar cleanup without changing routes, permissions, or the existing hamburger collapse behavior.

- Kept the collectZ mark visually in line with the collapsed navigation icons.
- Kept collapsed icon-only nav buttons accessible by preserving their text labels as `aria-label` values.
- Preserved the expanded sidebar header behavior from `3.16.19`.
- Left the hamburger/collapse affordance unchanged while the top-brand-as-collapse-control idea remains a product follow-up.
