## Summary

This patch replaces the always-visible Cards/List buttons in shared library headers with a compact view-mode menu whose trigger reflects the current layout.

## What changed

- Added a shared view-mode menu to the page header toolbar.
- The trigger shows the active layout icon and opens a small menu with Grid view and List view options.
- The selected layout is marked in the menu while preserving accessible menu semantics.
- Existing search, filter, sort, and add controls remain available.

## Breaking changes

None.

## Environment/config changes

None.

## Migration and data impact

No schema migration or data backfill is required. This patch changes shared frontend header presentation only.
