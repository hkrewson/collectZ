# collectZ v3.4.136

## Summary

This patch adds a read-only Plex full-library reconciliation preview.

## Changes

- Adds an admin-only `POST /api/media/plex-reconciliation-preview` endpoint.
- Classifies Plex library items into `alreadyLinked`, `wouldUpdate`, `wouldCreate`, and `conflict` buckets.
- Uses the current Plex library section paths without changing import behavior.
- Keeps the preview explicitly read-only: no collectZ import mutation and no Plex writeback.
- Adds a fake PMS runtime smoke proving bucket behavior and sanitized evidence.
- Keeps scheduled reconciliation automation and broad Plex import rewrites out of scope.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- Plex reconciliation preview remains admin-only and does not expose Plex tokens, token-bearing URLs, private IPs, or media file paths.
