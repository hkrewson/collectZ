# v3.4.48

## Event schedule Now / Next view

- Version: `3.4.48`
- Target milestone: `3.4.48 — Event Schedule Now / Next Read-Only View`
- Date: 2026-05-01

### What changed

- Added a compact Now / Next block to the Event drawer Catalog section.
- Shows the catalog session currently in progress when one exists.
- Shows the next upcoming catalog session when one exists.
- Shows a short later-today list for additional same-day catalog sessions.
- Marks catalog sessions that are already in the user's personal schedule.
- Keeps hidden and cancelled catalog sessions out of the Now / Next summary.

### Why it matters

The schedule catalog is now easier to use during an event. Instead of scrolling through the full catalog first, users can quickly see what is happening now and what starts next, while keeping catalog data separate from personal schedule plans.

### Security and privacy notes

- This patch does not add provider import automation, background sync, notifications, realtime location, presence, or offline mutation queues.
- The summary reads from existing event-scoped catalog/session-plan data.
- Personal Sched ICS feed URLs remain hidden and are not used as a full event catalog source.
