## Summary
This release tightens image and proof control language across drawer surfaces so cover images, event artifact images, and signature proof images use the same action vocabulary.

## What changed
### Image and proof control language
- Standardized source actions around `Choose from Library` and `Take Photo` where explicit source selection is shown.
- Aligned media cover copy with the shared image picker language: add/replace/remove cover now uses the same source hint as other image controls.
- Updated event artifact row actions to show clear `Open image` and `Remove image` labels instead of icon-only controls.
- Hid event artifact open/remove image actions while a replacement image is selected for that same edited artifact.

### Signature proof controls
- Renamed the proof picker to `Proof image` and the selected-file state to `Selected proof image`.
- Updated proof upload/remove actions to `Add proof image` and `Remove proof`.
- Made proof list links use `Open proof` while keeping proof title/type metadata visible beside the action.
