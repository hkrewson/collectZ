# collectZ v3.4.83

## Event Social Mobile Day-Of Summary

This patch continues the Event Social Planning Mobile Web Experience by making the mobile event social overview more useful during a con. The drawer now leads with a day-of summary that shows the current schedule focus, next meetup, who is involved, and quick jumps into the existing Schedule, Meetups, and People sections.

### What changed

- Reworked the mobile-only Event social overview into a day-of social plan summary.
- Shows schedule focus metadata as separate `When`, `Where`, and `Visibility` fields for faster phone scanning.
- Shows the next meetup with time, place, group context, and visibility without requiring the user to open the Meetups panel first.
- Keeps the people/group preview visible in the overview so users can quickly see who a plan involves.
- Adds compact quick-jump buttons that open and scroll to the existing Schedule, Meetups, and People sections.
- Preserves desktop planning views, Event-local attendee ownership, duplicate attendee guardrails, `Add me`, first-social-action auto-create, notification contracts, and manual non-user attendee entry.
- Keeps native companion behavior, push/Discord/email delivery, cross-event identity, external contacts, realtime presence, and broader friend-graph work out of scope.
