## Summary

This patch adds a compact mobile overview to Event plans so phone users can quickly see the next schedule item, next meetup, people, and groups before opening the detailed planning sections.

## User-Facing Changes

- Event detail drawers now show a phone-sized social overview above the Event plans accordions.
- The overview summarizes people, groups, meetups, the next schedule plan, and the next meetup.
- Private/group/shared planning labels remain visible in the mobile overview.
- Existing schedule, Sched feed, people, groups, and meetup management controls remain in place below the overview.

## Validation Notes

- This is a read-first mobile web overview patch only.
- No API/schema changes, full schedule catalog discovery, friend notifications, native companion behavior, location/presence tracking, or fast meetup editing was added.
- Docker-first frontend/backend rebuild should report `3.4.38` from `/api/health`.
