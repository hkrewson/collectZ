# collectZ v3.4.93

## Kavita Chapter-as-Issue Row Fan-out Implementation

This release implements the opt-in chapter-as-issue import behavior defined in `3.4.92`.

### Shipped

- Added default-off Kavita chapter fan-out for eligible comic/manga chapters.
- Preserved the existing series-level Kavita import as the default behavior.
- Created chapter issue rows with distinct `kavita:chapter:{chapterId}` provider identities and parent links back to the Kavita series row.
- Added an admin Kavita import checkbox for importing comic chapters as issue rows.
- Extended the Kavita import smoke to cover default import behavior, opt-in issue creation, repeat-sync idempotency, book exclusion, special-chapter skipping, cover proxy behavior, and secret-free launch URLs.

### Boundaries

- Kavita remains read-only.
- Embedded reading, reader page proxying, reading progress sync, metadata writeback, per-space Kavita administration, and shared provider abstractions remain future work.
