# v3.4.34

- Date: 2026-04-29
- Version: `3.4.34`
- Target milestone: `3.4.34 — Event Schedule Day Navigation and Now/Next Readability`

## Summary

This patch improves long Event schedules by adding lightweight day navigation, an Upcoming filter, and a local-time now/next affordance to the selected schedule agenda.

## User-Facing Changes

- Multi-day Event schedules can be filtered by day from the detail drawer.
- Upcoming selected sessions can be filtered without scrolling through already-past plans.
- When a current or upcoming selected session exists, the schedule can jump to that session.
- Past events remain readable without showing irrelevant now/next controls.
- The schedule agenda keeps the compact row treatment from `3.4.33`.

## Validation Notes

- This is a selected personal schedule readability patch only.
- No continued Sched syncing, full schedule catalog import, friend notification, or provider discovery behavior was added.
- Docker-first frontend/backend rebuild should report `3.4.34` from `/api/health`.
