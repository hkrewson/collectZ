## Summary

This patch removes the remaining `control plane` wording from the candidate public source surfaces. It is one step in making the public mirror useful enough to include frontend source and API contract files later.

## What changed

- Reworded the admin workspace management description in the frontend.
- Reworded four OpenAPI admin workspace summaries away from internal operations language.
- Reduced the public-source boundary audit from `32` findings across `6` files to `27` findings across `5` files.
- Confirmed the `control plane` blocker category is cleared from the audit.

## Deployment and verification

- Rebuild backend and frontend containers with `APP_VERSION=3.18.3`.
- Verify `/api/health` reports version/frontend/backend/build `3.18.3`.
- Verify Help > Releases serves `3.18.3`.
- Run `npm run audit:public-source-boundary` and confirm no `control plane` findings remain.
- Run `npm --prefix backend run test:openapi`.
- Run `npm run release:local-gate`.
