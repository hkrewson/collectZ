## Summary

This patch makes platform admins explicitly choose between workspace/library work and the platform control plane.

## What changed

- Added a Workspace/Platform mode switch to the sidebar for platform admins.
- Workspace mode shows collection work, library tools, import, Help, Workspace settings/integrations, and workspace review tools.
- Platform mode shows platform-only surfaces such as Platform Settings, Runtime, Platform Activity, All Workspaces, and All Members.
- Merge Review remains under Workspace and is hidden from Platform mode.
