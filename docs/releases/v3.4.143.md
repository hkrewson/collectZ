# collectZ v3.4.143

- Version: `3.4.143`
- Status: `released`

## Summary

This patch adds a durable Plex reconciliation conflict review queue so unsafe Plex sync matches can be resolved intentionally after automated sync.

## Changes

- Stores Plex reconciliation conflicts in a first-class review table instead of only embedding them in sync job summaries.
- Adds admin readback for open and resolved Plex conflict review rows.
- Adds explicit conflict resolution actions to create a separate local Plex-linked title or dismiss the conflict.
- Shows open Plex conflict reviews in the Plex integrations panel with direct resolution controls.
- Keeps Plex writeback disabled and leaves broad import rewrites out of scope.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- This change does not expose Plex tokens, raw Plex URLs, media file paths, or raw PMS source payloads in browser-visible conflict review responses.
