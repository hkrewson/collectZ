# collectZ v3.4.75

## Platform Companion Friend-Aware Session Changes

This patch adds a native-friendly session-change coordination contract to the Event companion snapshot. Platform clients can now discover selected-recipient schedule change behavior, supported intents, recipient policy, and endpoint hints directly from `/api/events/:id/companion/today`.

### What changed

- Added `friend_aware_changes` with contract version `event-companion-friend-aware-session-changes.v1`.
- Advertises supported intents such as `join`, `replace`, `meet`, `leave`, `backup`, and `status_update`.
- Exposes selected-recipient policy, preview-before-send guidance, and endpoint hints for preview, records, drafts, delivery-boundary readback, and inbox readback.
- Keeps privacy/visibility backend-owned and keeps push, email, device registration, global inboxes, and broad friend discovery out of scope.
