# collectZ v3.4.139

## Summary

This patch adds a temporary read-only Plex reconciliation review UI.

## Changes

- Adds a temporary reconciliation preview panel to the Plex integrations settings.
- Shows linked, would-update, would-create, and conflict bucket counts.
- Lets admins inspect sanitized bucket rows returned by the existing preview endpoint.
- Adds an option to queue the existing read-only reconciliation preview job and read back the stored result.
- Keeps reconciliation mutation and Plex writeback disabled.
- Adds browser coverage that proves the temporary UI displays buckets without an apply action.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- This change does not expose Plex tokens, raw Plex URLs, media file paths, or mutation controls.
