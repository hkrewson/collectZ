# collectZ v3.4.76

## Event Schedule Shared Session Presence Readback Polish

This patch makes shared session presence easier to read in the Event drawer. Shared attendance now shows up as clearer presence strips on schedule cards, with better people/group breakdowns when you expand a session.

### What changed

- Improved collapsed shared-presence readback on Now / Next, catalog session rows, and personal schedule rows.
- Added clearer count chips for visible people/groups and shared plan state.
- Expanded shared-attendance blocks now show named `People` and `Groups` sections when that data exists.
- Preserved the existing visibility-safe Event-local readback boundary.
- Kept notifications, push/email/device delivery, global inboxes, and friend-graph work out of scope.
