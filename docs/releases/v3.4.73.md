# collectZ v3.4.73

## Event Schedule Delivery Attempt Readback UI

This patch makes Event schedule notification delivery-attempt audit records visible in the Event drawer.

### What's New

- Added compact delivery-attempt summaries to sent schedule notification records.
- Added delivery-attempt readback inside notification history.
- Shows the local recipient/group, attempt status, and completed time when attempt rows are available.
- Keeps the UI language clear that these are Event-local audit records, not push, email, native device, realtime, or external provider delivery.
- Updated browser coverage and source contract checks for the readback UI.

### Notes

This remains a local coordination and audit workflow. Provider settings, push/email delivery, native device registration, and global notification inboxes remain future work.
