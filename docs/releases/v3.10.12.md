## Summary

This patch refines the Capture Inbox funnel menu by changing the cramped horizontal Review tabs into a compact two-column radio-style picker.

## What changed

- Kept Capture and Source as simple selects inside the funnel menu.
- Replaced the Review tab strip inside the funnel with a two-column choice grid.
- Preserved the same review filters and API behavior.
- Updated browser regression coverage to expect the Review filter as radio-style choices.
