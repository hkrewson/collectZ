# collectZ v3.4.119

Release date: 2026-05-07

## Summary

This release adds a revocable Plex Now Playing display link. Admins can generate a limited token for `/now-playing?token=...` so a passive display can show the sanitized Plex Now Playing viewer without an admin browser session.

## Changes

- Added hashed, revocable Plex Now Playing display-token metadata to integration settings.
- Added admin generate/revoke actions for the display link in Plex Integrations.
- Added token-only viewer and image proxy endpoints that can read only sanitized now-playing data and Plex-relative image assets.
- Updated `/now-playing` so token links bypass the login flow while keeping the existing admin-session viewer intact.
- Kept Plex imports, webhooks, watch-state writes, and queue/next-up behavior out of scope.
