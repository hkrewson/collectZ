# collectZ v3.4.121

Release date: 2026-05-07

## Summary

This release adds a vertical poster-only layout option for Plex Now Playing displays. It keeps the existing saved display preferences and display-token flow intact while giving passive screens a clean full-height poster mode.

## Changes

- Added a saved Plex Now Playing display layout preference.
- Added a `Standard` versus `Vertical poster only` layout selector in Plex Integrations.
- Updated `/now-playing` to render poster-only mode as a full-screen portrait poster surface.
- Returned the normalized layout mode from admin-session and display-token Now Playing endpoints.
- Kept Plex imports, display-token generation/revoke behavior, image proxying, webhooks, scheduled sync cadence, and watch-state work unchanged.
