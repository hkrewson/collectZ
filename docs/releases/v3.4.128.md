# collectZ v3.4.128

## Summary

This patch adds the read-only Plex watch-state sync contract that future watched/progress automation will build on.

## Changes

- Added a Plex watched-state contract with default and minimum cadence expectations.
- Added safe normalization for Plex completed, in-progress, and unwatched states from metadata and episode leaf readbacks.
- Added a fake-PMS smoke proof that reads watched-state fields without calling Plex scrobble or unscrobble writeback paths.
- Kept collectZ watched-state mutation and Plex watched-state writeback as future explicit opt-in work.
