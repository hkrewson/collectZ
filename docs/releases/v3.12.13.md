## Summary

This patch lets Dashboard Review sparse-metadata rows use the same assisted lookup flow as missing-identifier rows, while keeping manual fallback fields available.

## What changed

- Sparse-metadata review drawers now show media-specific provider lookup controls.
- Sparse-metadata drawers keep `Manual fallback` fields open by default.
- Applying a lookup candidate can stage title, year, cover, provider identity, or type-specific metadata through the existing pending-update readback.
- Save readiness, defer, dismiss, and drawer-only review behavior are unchanged.
