## Summary

Kavita comic imports now lean toward showing the issues Kavita exposes, not quietly stopping at the parent series row. The admin Kavita import control defaults chapter issue fan-out on, special chapters import as issue rows, and chapter rows can be kept when Kavita provides sort/order metadata even if the usual issue fields are sparse.

## Changed

- Defaulted the Kavita admin import checkbox to import comic chapters as issue rows.
- Imported Kavita comic special chapters as issue rows instead of skipping them.
- Used Kavita chapter `sortOrder` as a fallback issue/order signal.
- Preserved Kavita chapter sort order and special markers in imported type details.
- Updated the Kavita import/sync smoke to prove special chapter import behavior.
