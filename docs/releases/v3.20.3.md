## Summary

This patch makes Plex initial import a deliberate Sync workflow. The Plex Sync tab now shows selected libraries and media types before queuing import work, and the old global Plex import button has been removed.

## What changed

- Added a compact `Initial import` block to Settings > Integrations > Plex > Sync.
- Shows selected Plex libraries from Setup before starting an import.
- Shows inferred media types for selected Plex sections when detected library metadata is available.
- Warns about saved section IDs that are not currently detected.
- Prevents queuing a Plex import when no library sections are selected.
- Removed the duplicate bottom-level `Import from Plex` action from the integration page.
