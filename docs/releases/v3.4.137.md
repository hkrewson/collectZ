# collectZ v3.4.137

## Summary

This patch adds a queued Plex reconciliation preview job.

## Changes

- Adds admin-only `POST /api/media/plex-reconciliation-preview/run`.
- Stores read-only reconciliation preview results in `sync_jobs` history.
- Preserves the `alreadyLinked`, `wouldUpdate`, `wouldCreate`, and `conflict` bucket model.
- Keeps Plex writeback and collectZ import mutation disabled for the job.
- Extends the fake PMS reconciliation smoke to prove queued job readback through `/api/media/sync-jobs/:id/result`.
- Keeps automatic reconciliation mutation, UI controls, and broad Plex import rewrites out of scope.

## Security vulnerability triage

- High findings: none identified locally; dependency preflight remains the local source for dependency audit status.
- Owner: collectZ maintainers.
- Target remediation milestone: none required for this release.
- Plex reconciliation preview jobs remain admin-only and do not expose Plex tokens, token-bearing URLs, private IPs, or media file paths.
