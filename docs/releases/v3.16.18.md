## Summary

This patch tightens the sidebar underline interaction so selected items use the same bright blue as tab selection, while account menu rows use the same quieter hover treatment as the main navigation.

## What changed

- Fixed selected sidebar underlines so the active state uses the bright app blue instead of inheriting the muted hover color.
- Kept inactive sidebar hover/focus underlines muted.
- Removed the distracting active/focus outline from the account trigger when the profile menu is open.
- Updated profile menu rows to use text hover plus muted underline feedback instead of gray block hover fills.
- Kept the active profile row on a solid blue underline when the profile page is selected.
