- Status: completed

## Summary

This patch makes the existing Library search box identifier-aware. Users can search by stored UPC/EAN/ISBN-style identifiers, TMDB ids, provider ids, Plex keys, and identity-alias metadata without leaving the current library surface.

## What changed

- Extended `GET /api/media` search to match stored UPC, ISBN-style type details, TMDB ids, provider ids, Calibre/Kavita ids, Plex metadata keys, and identity aliases.
- Guarded numeric-normalized matching so non-numeric searches do not become broad empty-string identifier matches.
- Updated the Library search placeholder to advertise identifier-aware search.
- Added unit/source coverage for the identifier search contract.
