## Summary

This patch moves backup/export readback into Workspace Settings while keeping Platform Settings clearly scoped to instance-wide backup/export.

## What changed

- Added workspace-scoped portability status and export endpoints under `/api/spaces/:id/portability`.
- Workspace Settings now shows `Workspace backup and portability` with scope readback for the selected workspace.
- Workspace JSON/CSV exports reuse the existing redacted portability export behavior while filtering database rows to the selected workspace.
- Platform Settings keeps its existing platform-wide export surface.
- Portability status and export manifests now include explicit `scope` metadata.
