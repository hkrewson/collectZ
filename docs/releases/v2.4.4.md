## Summary
This release closes the Events + Collectibles rollout with sorting/filtering improvements and production-safe flag gating.

## What changed
- Added sort toggle controls to Events and Collectibles views.
- Updated Events search to cover title and location from a single search field.
- Removed separate Events location filter input from the UI.
- Standardized Events ordering by start date and end date with stable fallbacks.
- Added feature flags:
  - `events_enabled`
  - `collectibles_enabled`
- Gated Events and Collectibles API routes behind feature flags.
- Gated Events and Collectibles navigation/views behind feature flags.
- Added migration `v38` to seed new feature flags with safe behavior for existing installs.

## Breaking changes
- None.

## Environment/config changes
- New optional env overrides:
  - `FEATURE_FLAG_EVENTS_ENABLED`
  - `FEATURE_FLAG_COLLECTIBLES_ENABLED`

## Migration and data impact
- Added migration `v38`: `Add feature flags for Events and Collectibles library surfaces`.
- Existing populated installs preserve behavior by enabling both flags.
- New/empty installs default both flags to disabled.
- No destructive schema/data operations.
