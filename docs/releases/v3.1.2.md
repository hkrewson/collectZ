## Summary
This patch release tightens the post-split shell experience without reopening shared-core architecture work. The auth page now uses a simpler single-column layout, and the Help and Help Admin surfaces carry less narration and less panel-heavy chrome, so platform and homelab feel more consistent and less overdesigned.

## What changed
### Auth surface cleanup
- Reworked the auth page into a single-column layout with one clear center of gravity instead of the earlier split-screen composition.
- Simplified auth copy so the page no longer reads like product explanation or onboarding theater.
- Removed the unintended boxed login-panel treatment and kept the form shell lighter.

### Help and Help Admin cleanup
- Reduced panel-heavy composition across Guidance, Releases, Metrics, and Support areas.
- Removed explanatory helper copy that described the interface instead of letting the layout speak for itself.
- Tightened thread, timeline, and empty-state chrome so the support surface feels calmer and more native to the rest of the app.

### Edition consistency
- Synced platform and homelab frontends so both running editions now serve the same updated auth and help surfaces instead of drifting due to stale frontend containers.
