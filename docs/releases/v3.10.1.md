- Status: `completed`

## Summary

This patch tightens the mobile search and action areas across the main library surfaces so Library, Collectibles, Art, and Events feel more usable on phone-width screens.

## What changed

- Made mobile library toolbars more compact without introducing a new visual pattern.
- Let Library search, quick filters, view toggles, sort, and add actions share predictable mobile rows.
- Collapsed mobile title/action rows so view mode, sort, and add no longer consume a separate toolbar band.
- Paired search with quick filters where possible, and kept bulk select controls out of the mobile header chrome.
- Hid long library description copy on phone-width Collectibles, Art, and Events headers so controls start sooner.
- Kept Events date filters visible while tightening their row layout.
- Added a browser regression that checks mobile Library, Collectibles, and Events toolbars do not overflow and stay within expected compact heights.
