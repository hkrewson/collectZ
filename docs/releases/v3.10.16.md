## Summary

This patch consolidates library filter controls into a single funnel menu so desktop and mobile pages stop carrying loose secondary filter fields in the main search toolbar.

## What changed

- Added a shared `FilterMenu` primitive using the existing funnel icon.
- Moved media library quick filters into the funnel menu.
- Moved Art event and exclusive filters into the funnel menu.
- Moved Collectibles category, event, and exclusive filters into the funnel menu.
- Moved Events date range filters into the funnel menu.
- Kept live search, view mode, sort, and add actions outside the filter menu.
- Left Loans status tabs unchanged because they act as primary loan state tabs.

## Breaking changes

None.

## Environment/config changes

None.

No schema migration or data backfill is required. This patch changes frontend presentation, browser regression coverage, version metadata, and release feed data only.
