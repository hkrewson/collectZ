## Event schedule join leave replace actions

- Version: `3.4.64`
- Date: `2026-05-02`
- Target milestone: `3.4.64 — Event Schedule Join Leave Replace Actions`

### Summary

This release adds explicit Event schedule intent actions so users can join, leave, mark backup, or replace conflicting catalog sessions from session cards without hunting through status menus.

### Changes

- Added compact `Join`, `Leave`, `Backup`, and `Replace with this` actions to catalog session cards.
- Wired `Replace with this` into the existing conflict-update path so conflicting plans move to backup.
- Kept the existing plan-state select for precise manual editing.
- Added matching quick intent controls to expanded personal schedule rows before save/preview.
- Added browser/unit coverage for the join, leave, backup, and replace flow.
