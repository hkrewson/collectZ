## Summary

This release continues the maintainability guardrails work where the current quality report shows clear value: `AdminIntegrationsView.jsx` was the top remaining frontend lint pocket after `3.23.11`. The slice clears Admin Integrations import, hook-baseline, helper-order, dependency, and boolean-cast findings without changing integration settings behavior.

## What changed

- Removed the stale default React import from the Admin Integrations view.
- Documented intentional integration-section synchronization from external dashboard routes and runtime feature availability.
- Documented workspace-scoped runtime feature-flag reset behavior.
- Stabilized Plex scheduler, readback, and conflict-review refresh helpers with `useCallback` before the effect that invokes them.
- Removed redundant boolean casts from the log export debug control.
- Reduced the full frontend quality baseline from `44` errors / `29` warnings to `37` errors / `27` warnings.
