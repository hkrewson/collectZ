# collectZ v3.4.149

- Version: `3.4.149`
- Status: `released`

## Summary

This patch corrects the Plex provider import migration contract. `/media/providers` is now treated as capability discovery, while future import migration resolves the official Plex library provider's advertised documented `/library/...` paths before any import behavior changes.

## Changes

- Adds a provider-advertised import path contract for the official Plex library provider.
- Recognizes `/library/sections/all` as the advertised sections root when Plex exposes it.
- Keeps `/library/sections` as the compatibility fallback for current Plex installs.
- Adds a fake-PMS smoke proof that the contract is read-only, token-safe, and does not mutate imports.
- Keeps Plex import, reconciliation, webhooks, watched-state writeback, rating writeback, and automatic sync behavior unchanged.
