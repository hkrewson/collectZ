## Summary

This patch upgrades the frontend Tailwind build pipeline to Tailwind CSS 4 and uses the dedicated Vite plugin while preserving the existing collectZ design tokens and component CSS.

## What changed

- Updated frontend `tailwindcss` from `^3.4.0` to `^4.3.0`.
- Added `@tailwindcss/vite` and wired it into the Vite plugin list.
- Replaced the old `@tailwind` CSS directives with the Tailwind 4 import plus a compatibility `@config` reference to the existing JavaScript Tailwind config.
- Removed the obsolete PostCSS Tailwind/autoprefixer pipeline.
- Flattened local component `@apply` recipes that chained through custom CSS classes so Tailwind 4 can compile them.

## Breaking changes

None expected for users.

## Environment/config changes

None for operators. This changes the frontend build-time CSS toolchain only.

## Migration and data impact

No schema migration or data backfill is required. This patch changes frontend styling dependencies and build configuration only.
