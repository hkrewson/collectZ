## Summary

This patch finishes a small post-Vite dependency hygiene pass by removing stale CRA/Webpack-era frontend package overrides that no longer apply to the current Vite dependency tree.

## Changes

- Removed the ignored local `frontend/build/` CRA output directory from the workspace.
- Removed frontend package overrides for packages no longer present in the current lockfile, including old ESLint, Webpack, Terser, Istanbul, and CRA-adjacent override targets.
- Kept the active `follow-redirects` override because it is still present through Axios.
- Refreshed frontend lockfile metadata with a current Node/npm runtime.
- Added a source-contract assertion that the frontend override set stays limited to active, intentional entries.

- None intended.

- Local stale `frontend/build/` output is not used by the app; Vite output remains `frontend/dist/`.
