# v2.7.3

## Version and date
- Version: `2.7.3`
- Date: `2026-03-20`

## Milestone target and status
- Target milestone: `2.7.x maintenance lane`, `2.7.3` frontend dev/build advisory cleanup
- Status: `Completed`

## Summary
This patch release reduces the remaining frontend development-tooling advisory surface without reopening the legacy CRA/react-scripts toolchain. It takes the low-risk lockfile and override fixes that are compatible with the current frontend build stack and leaves the remaining CRA-coupled advisories for a later, more explicit toolchain-focused maintenance slice.

## What changed
### Frontend dependency maintenance
- Updated the frontend `flatted` override to `3.4.2`.
- Updated `@tootallnate/once` to `3.0.1` through the `http-proxy-agent` path used in the Jest/jsdom dev chain.
- Updated all active `serialize-javascript` paths used by the frontend build chain to `7.0.4`.
- Reworked the `bfj` / `jsonpath` / `underscore` path so the old `jsonpath` advisory family is removed from the active frontend development tree.

### Backend
- No functional backend changes beyond version sync to `2.7.3`.

### Docs and release evidence
- Added a new release note for `2.7.3`.
- Refreshed local dependency-audit and SBOM artifacts.
- Recorded a new local preflight result in `preflight-go-no-go.md`.
