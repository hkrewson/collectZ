## Summary

This patch makes Event meetups easier to manage during planning or day-of-con use by adding compact in-place controls for meetup status and quick notes.

## User-Facing Changes

- Meetup rows in Event plans can now expand into a small status and notes editor.
- Meetup status can be changed using the existing backend values: planned, tentative, cancelled, or done.
- Meetup notes can be added or replaced without leaving the Event drawer.
- Existing meetup add/remove behavior and the mobile social overview remain intact.
