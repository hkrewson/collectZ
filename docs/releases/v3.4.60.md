## Event schedule notification inbox and recipient readback

- Version: `3.4.60`
- Date: `2026-05-02`
- Target milestone: `3.4.60 — Event Schedule Notification Inbox and Recipient Readback Contract`

### Summary

This release turns Event-local schedule notifications into recipient-readable records inside the Event drawer. Sent local notices now create recipient readback rows, expose an Event-local inbox, and support marking recipient rows read or acknowledged.

### Changes

- Added normalized Event-local schedule notification recipient records.
- Added recipient inbox/readback API endpoints for sent local schedule notifications.
- Added read and acknowledged state for recipient rows.
- Added a compact Event drawer Notification inbox section.
- Kept the boundary explicit: this is local readback only, not push, email, device delivery, or a global friend inbox.
