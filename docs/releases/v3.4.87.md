# collectZ v3.4.87

## Kavita Metadata Mapping Detail Foundation

This patch enriches Kavita imports with native library and series metadata so imported digital-library rows are easier to classify, inspect, and safely re-sync.

### What changed

- Added durable Kavita detail fields for library id/name/type, series id/name/localized/original/sort names, format, pages, and cover source.
- Improved book versus comic classification by using Kavita library type before falling back to name hints.
- Expanded the Kavita import smoke to import a book and comic library side by side.
- Preserved the `3.4.86` behavior that safely reuses matching non-Kavita titles and keeps repeat syncs idempotent.
- Kept metadata writeback, embedded reading, reading progress, and provider framework cleanup out of scope.
