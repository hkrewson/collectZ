# v3.4.67

## Event schedule recipient selection UI polish

- Version: `3.4.67`
- Date: `2026-05-02`
- Target milestone: `3.4.67 — Event Schedule Recipient Selection UI Polish`

### Summary

This release makes Event-local schedule notice recipients visible and editable before a draft is saved or a local notice is recorded.

### Changes

- Added a compact recipient selector to the schedule notification composer.
- Seeded selected people and groups from the existing share preview.
- Let users uncheck eligible people/groups before saving or sending the Event-local notice.
- Sent selected recipients through the existing `recipient_attendee_ids` and `recipient_group_ids` contract.
- Disabled sending when a preview is private or no recipients remain selected.
- Kept push, email, native device delivery, global inboxes, friend graphs, and realtime social behavior out of scope.
