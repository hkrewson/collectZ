# v3.4.65

## Event schedule change notification templates

- Version: `3.4.65`
- Date: `2026-05-02`
- Target milestone: `3.4.65 — Event Schedule Change Notification Templates`

### Summary

This release adds action-aware local notice templates for Event schedule changes, so join, leave, backup, and replace flows start with useful default wording while still staying selected-recipient and Event-local.

### Changes

- Added a backend-owned `message_intent` hint to schedule change previews and local notification records.
- Added template defaults for join, leave, replace, backup, and general status updates.
- Show suggested local-notice copy in the schedule change preview.
- Persist the selected template copy when saving drafts or sending local notices.
- Kept notification behavior Event-local only with no push, email, device, global inbox, or broadcast delivery.
