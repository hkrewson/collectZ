## Summary

This patch makes the authoritative CodeQL baseline match the clean GitHub Actions checkout by excluding generated, dependency, report, coverage, build, and local SARIF output from maintained-source analysis.

## What changed

- Added a repository CodeQL configuration for the maintained-source baseline.
- Wired the CodeQL workflow to load the configuration while preserving JavaScript/TypeScript analysis and `security-extended,security-and-quality`.
- Kept `AlertSuppression.ql` and the local collectZ JavaScript model pack documented for exploratory local CLI runs instead of passing CLI-only specifiers to the hosted CodeQL Action.
- Added the local model pack manifest and request-forgery barrier model for exploratory CLI parity runs.
- Documented the operating rule that generated-artifact findings are exploratory triage noise unless they trace back to maintained source.
- Added unit-test coverage for the CodeQL workflow/config/model-pack wiring.

## Breaking changes

- None.

## Environment/config changes

- GitHub CodeQL advanced setup now loads `.github/codeql/codeql-config.yml`.
- The hosted workflow uses GitHub-supported `queries: security-extended,security-and-quality`; local CLI-only `AlertSuppression.ql` and model-pack options stay out of the authoritative parity command.
- Authoritative CodeQL analysis excludes generated/noisy paths including `artifacts/**`, `backend/artifacts/**`, `frontend/artifacts/**`, Playwright report and test-result folders, coverage output, build/dist output, dependency folders, and local SARIF files.

## Security vulnerability triage

- Authoritative CodeQL baseline is the GitHub Actions clean checkout of committed maintained app source.
- Local exploratory CodeQL runs may include uncommitted/generated files, but those findings are triage noise unless they point back to maintained source.
- Raw local SARIF result counts are not expected to exactly match GitHub code-scanning alert counts because GitHub fingerprints, deduplicates, branches, and suppresses alerts in the hosted alert model.
