# collectZ v3.20.6

Plex writeback is now explicitly opt-in.

## Highlights

- Added persisted Plex rating and watched-state writeback switches.
- Added compact controls in Settings > Integrations > Plex > Sync.
- Hid Library drawer writeback actions unless the matching switch is enabled.
- Added backend guards so direct API calls cannot push to Plex while disabled.

## Notes

Both writeback options default to off. Manual import, reconciliation, webhook hints, and conflict review behavior are unchanged.
