## Summary
This release starts the drawer-shell cleanup by moving Media, Art, Collectibles, and Events detail drawers onto one shared slide-over shell while tightening the most obvious mobile spacing drift.

## What changed
### Shared detail drawer shell
- Added a shared `DetailDrawerShell` primitive for the standard overlay, side panel, border, and slide-in wrapper.
- Migrated Media, Art, Collectibles, and Events detail drawers to the shared shell.
- Kept edit drawers out of scope so this remains a small, rollback-safe detail-view pass.

### Mobile density pass
- Reduced mobile-only header/body padding in Media, Collectibles, and Events detail drawers while preserving desktop spacing.
- Kept Art's existing compact mobile detail treatment intact.
- Added source assertions for shared shell usage and the mobile spacing guardrails.
