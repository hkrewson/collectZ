## Summary
This patch release makes library detail drawers feel more content-shaped and less template-driven after the earlier library and profile cleanup passes. Books, collectibles, events, comics, TV seasons, and the shared media drawers now use calmer hierarchy, quieter actions, and more truthful source treatment, while newer library tabs also persist correctly across refresh.

## What Changed
### Book drawer cleanup
- Reworked book drawers into a wider, less machine-shaped layout with calmer title treatment and labeled source links.
- Reduced provider-plumbing exposure so raw URLs, duplicated destinations, and low-value technical metadata no longer dominate the book detail surface.
- Tightened the lower-half arrangement for editions, rating, and notes so the drawer feels less bottom-light.

### Collectible and event drawer restraint passes
- Flattened collectible and event drawers rather than redesigning them into heavier panel systems.
- Replaced louder button-like external actions with quieter links where appropriate and toned down footer controls.
- Tightened spacing and information hierarchy so these sparser drawers feel denser and more intentional without becoming overdesigned.

### Shared media drawer normalization
- Normalized shared media drawers for movies, TV, games, and comics so titles, notes, external actions, and footer controls no longer feel like leftovers from the older button-heavy drawer system.
- Corrected provider/source labeling so games point to `View on IGDB` instead of inheriting movie/TV labels, and media actions read more truthfully by destination.

### Comic drawer cleanup
- Suppressed visible internal and provider IDs in comic drawers.
- Stopped leaking Calibre and OPDS plumbing into `Type Details` and converted provider links into clearer labels such as `View on Metron` and `Download on Calibre`.
- Added a restrained long-overview treatment so oversized comic summaries collapse behind a simple show more/show less toggle.

### TV season cleanup and shared-library persistence fix
- Replaced stacked nested TV season cards with a flatter season-tab treatment using shorter `S1`-style labels and quieter watched-state feedback.
- Fixed newer library tabs like events and collectibles so they persist correctly across refresh instead of falling back before feature flags finish loading.
