## Summary
This release consolidates the repeated drawer backdrop/header image treatment into one shared frontend primitive so Media, Art, Collectibles, and Events stay visually aligned.

## What changed
### Shared drawer backdrop primitive
- Added a shared `DrawerBackdrop` primitive for cropped drawer header images.
- Media detail drawers keep the `backdrop_path || poster_path` fallback introduced in `3.4.23`.
- Art, Collectibles, and Events now use the same primitive for their uploaded-image backdrop bands while preserving their existing sizing.
- Source assertions now guard the shared primitive wiring across the drawer families.

## Breaking changes
- None intended. This is a frontend implementation cleanup with preserved layout behavior.
