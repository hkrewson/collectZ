## Summary

This patch adds practical edition and variant editing to Library drawers using the reusable collectible trait contract.

## What changed

- Added a compact Edition drawer section for Library records.
- Edition controls now change by media type instead of using one generic variant field.
- Books support edition, printing, publisher line, ARC, first edition, limited release, and numbered-run details.
- Comics support variant cover, printing, run context, newsstand/direct edition, limited release, and numbered-run details.
- Games support platform, region, release line, collector edition, limited release, and promo/demo state.
- Movies, TV, and Audio support package, release, pressing, screener/promo, and limited-release style details.
- Saved edition details appear through existing collectible trait readback and compact trait pills.
