# collectZ v3.4.77

## Event Social Discovery Readback Polish

This patch makes the Event drawer’s social sections easier to scan. People, Groups, and Meetups now read back more of their related context directly, so it is easier to see who belongs with whom and what is coming up next without bouncing between sections.

### What changed

- Improved attendee cards with related-group, next-meetup, and next shared-plan readback.
- Improved group cards with member preview plus upcoming meetup/shared-plan cues.
- Improved meetup rows with clearer group, people, and notes context before opening edit controls.
- Kept the existing Event-local backend model and visibility-safe behavior intact.
- Kept friend-graph, realtime presence, push/email/device delivery, and native companion UI work out of scope.
