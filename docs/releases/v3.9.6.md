## Version and date

- Version: `3.9.6`

- Milestone: `3.9.6 — Apple/iTunes Wishlist Target Price Hits`

## Milestone target and status

Status: Complete.

This patch adds a reviewable Apple/iTunes wishlist readback for saved items whose refreshed store price is at or below the user's target price.

## Changes

- Added an authenticated `GET /api/wishlist/apple-itunes/target-price-hits` endpoint.
- Used the latest price-history snapshot when available, with current row price metadata as a fallback.
- Kept the readback scoped to the active workspace/library context.
- Added a compact Target price hits section to Library > Wishlist.
- Documented the target-price-hit response contract in OpenAPI.

## Security and privacy

- No new external provider calls, secrets, or writeback behavior are introduced.
- The new endpoint is authenticated and uses the existing Wishlist scope controls.
- Target price hits use existing Apple/iTunes wishlist rows and stored price snapshots only.
