## Kavita Read-Only Progress Visibility

## Summary

This patch adds the first read-only Kavita progress visibility path. Chapter-backed Kavita rows can ask collectZ to read current Kavita chapter progress and show a small progress panel in the media detail drawer.

## What Changed

### Shipped

- Added a scoped backend read endpoint for Kavita chapter-backed media rows.
- Read progress through the workspace-owned Kavita connection using `GET /api/Reader/get-progress`.
- Normalized progress readback to page/chapter/series/volume/scroll/timestamp fields.
- Added a compact read-only Kavita Progress panel to the media detail drawer for chapter-backed rows.
- Extended the fake-Kavita import/sync smoke to prove read-only progress readback and secret-free browser responses.
