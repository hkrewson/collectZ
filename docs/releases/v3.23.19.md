## Summary

This patch keeps long collectible-detail labels inside narrow Art poster cards. On iPhone-sized screens, signing and event-acquisition labels now truncate within their own card instead of overlapping the neighboring card or widening the page beyond the viewport.

## What changed

- Allowed shared poster cards to shrink below the intrinsic width of their metadata.
- Constrained trait-pill containers and individual pills to the available card width.
- Preserved the existing two-column mobile Art grid and full trait text through the pill title/readback.
- Added a mobile browser regression that checks both card-boundary containment and document viewport width with a deliberately long signer label.
