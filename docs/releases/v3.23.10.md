## Summary

This release continues the maintainability guardrails work where the current quality report shows clear value: `App.jsx` was the top remaining frontend lint pocket after `3.23.9`. The slice clears app-shell import, empty-catch, hook-baseline, and dependency findings without changing routing, auth, workspace scope, feature flags, dashboard tab, or now-playing behavior.

## What changed

- Removed the stale default React import from the app shell.
- Kept local logout cleanup explicit when the backend logout request has already expired.
- Documented intentional app-shell state synchronization effects for route normalization, backend-owned scope refresh, feature flag loading, retired tab redirects, feature-gated dashboard destinations, manageable workspace switching, and edition-safe dashboard tab selection.
- Included support-staff edition state in the safe-tab effect dependencies.
- Reduced the full frontend quality baseline from `60` errors / `32` warnings to `52` errors / `30` warnings.
