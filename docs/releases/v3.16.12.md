## Summary

This patch removes the redundant Workspace overview block so Workspace opens directly on the settings sections that users can act on.

## What changed

- Removed the large Workspace name, role, library, accessible workspace count, and visible library count block from the Workspace settings surface.
- Removed the old in-page workspace selector from Workspace settings.
- Kept workspace switching in the profile menu, where the app now treats Workspace and Platform as explicit contexts.
- Kept Workspace Settings, Backup, Export, Integrations, People, and Activity tabs unchanged.
- Updated browser coverage so it expects the useful Workspace sections instead of the old overview header.
