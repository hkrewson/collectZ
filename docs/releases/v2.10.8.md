## Summary
This patch release tones down the library shell navigation so it keeps its hierarchy and space context without feeling over-styled. The sidebar no longer leans on uppercase eyebrows, badge-like role chrome, gold active rails, or decorative sub-item bullets to explain itself, and instead reads more like ordinary product navigation.

## What changed
### Sidebar ornament reduction
- Removed the active gold accent rail from sidebar navigation items.
- Removed gold active icon tint from the main Library, Admin, and Global group buttons.
- Removed sub-item dot bullets from nested navigation links.
- Relaxed nested link typography so sub-navigation reads like normal navigation instead of tiny utility labels.

### Library context cleanup
- Changed the `Library` eyebrow label from uppercase tracked styling to plain text.
- Replaced the membership role badge treatment with quieter inline text while preserving the same context.

### Shell direction
- Preserved the sidebar’s structure, group expansion, and wayfinding behavior while reducing decorative dashboard chrome.
- Kept intentional functional cues such as numeric item badges and the sidebar brand mark.

## Breaking changes
- No backend or API-breaking changes are introduced in `2.10.8`.
- This release intentionally softens sidebar visual emphasis:
  - the nav no longer uses the previous gold active-rail/icon treatment,
  - nested items no longer use decorative bullets.
