## Summary

This patch makes Event schedule plans easier to adjust from the drawer by adding compact in-place controls for planning status, sharing visibility, and notes.

## User-Facing Changes

- Schedule rows in Event plans can now expand into a small editor.
- A schedule plan can be marked planned, maybe, backup, skipped, or attended.
- A schedule plan can stay private or be shared with the event workspace.
- Plan notes can be updated without leaving the Event drawer.
- Manually refreshed personal Sched ICS rows keep user-owned status, visibility, and notes while source details refresh.
