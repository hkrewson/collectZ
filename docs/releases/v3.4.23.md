## Summary
This release fixes clipped cover art in media detail drawers by giving every media drawer the same header breathing room that movie drawers already had.

## What changed
### Consistent drawer header spacing
- Audio, Books, Comics, Games, Movies, and TV detail drawers now always render a stable header band.
- When a TMDB backdrop is unavailable, the drawer reuses the item cover/poster image as the cropped backdrop behind the header.
- The cover/poster preview still renders as the foreground preview card, preserving the established movie drawer pattern.
- Source-level and focused browser regressions now check that the header band does not depend only on `backdrop_path`.
