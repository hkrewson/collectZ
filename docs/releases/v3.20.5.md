## Summary

This patch adds Plex reconciliation review filters. The Plex Sync tab can switch between open and resolved conflicts, filter loaded reviews by match reason, and read resolved review outcomes without reopening them.

## What changed

- Added Open and Resolved filters to Plex conflict review.
- Added match-reason filtering for loaded Plex conflict rows.
- Shows filtered and total conflict counts.
- Displays resolved conflict outcomes without action buttons.
- Kept existing conflict resolution behavior unchanged.
