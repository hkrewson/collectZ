### Summary

This release adds a compact All/Mine filter to the Event drawer Notification inbox so users can focus on Event-local schedule notifications linked to their own attendee identity.

### Changes

- Added an All/Mine filter to the Event-local Notification inbox.
- Wired the Mine filter to the existing `recipient=me` inbox contract.
- Kept the inbox language explicit that these are local readback records, not push, email, or device delivery.
- Added browser coverage for switching between all recipient rows and the current user's linked recipient rows.
