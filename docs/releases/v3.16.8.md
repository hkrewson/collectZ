This patch applies the compact navigation menu top-row treatment to the expanded desktop sidebar as well as the mobile navigation menu.

- Kept the desktop `collectZ` wordmark visible, but inside the same compact row.
- Kept the mobile menu top row compact from `3.16.7`.
- Added desktop-width browser coverage so the expanded sidebar top row cannot silently grow back.
