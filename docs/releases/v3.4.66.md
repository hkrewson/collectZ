## Event schedule template picker and message edit UI

- Version: `3.4.66`
- Date: `2026-05-02`
- Target milestone: `3.4.66 — Event Schedule Template Picker and Message Edit UI`

### Summary

This release lets users choose and edit the Event-local schedule notification message before saving a draft or sending a local notice.

### Changes

- Added a compact template picker to the schedule notification preview area.
- Added editable message text seeded from the selected template.
- Added a `meet` template intent for “Meet outside this room” coordination.
- Persist edited message copy through the existing Event-local notification record.
- Kept recipients selected from the existing preview and avoided broadcast defaults.
- Kept push, email, device delivery, global inboxes, and realtime social behavior out of scope.
