## Summary

This patch separates workspace backup readback from workspace export actions and shortens the Workspace navigation label to `Settings`.

- Renamed the Workspace navigation child item from `Settings & integrations` to `Settings`.
- Kept Workspace Integrations as its own workspace tab.
- Moved workspace backup readback into a dedicated `Backup` tab.
- Moved workspace export downloads and coverage into a dedicated `Export` tab.
- Removed backup/export readback from the general Workspace Settings tab.
- Kept the existing workspace portability API and export behavior unchanged.
